# 👾 IP Invaders

Un juego arcade educativo inspirado en los clásicos *Space Invaders*, diseñado para aprender conceptos de redes como direcciones IP, clases de red y subnetting mientras juegas.

---

## 🎮 Demo

> Abre el archivo `index.html` en tu navegador y empieza a jugar directamente.  
No requiere instalación ni servidor.

---

## 🧠 Concepto

En lugar de disparar a enemigos normales, en **IP Invaders** debes destruir la respuesta correcta a cada dirección IP que aparece en pantalla.

Aprenderás jugando conceptos como:

- IP privadas y públicas
- Loopback (127.0.0.1)
- Clases de red (A, B, C)
- Subnetting básico

## 📚 Objetivo educativo

Este proyecto está diseñado para ayudar a estudiantes a aprender conceptos básicos de redes de forma visual e interactiva, combinando:

- Aprendizaje activo
- Gamificación
- Repetición práctica

---

## 📈 Sistema de progresión

El juego incrementa la dificultad automáticamente:

1. 🟢 Nivel básico → IP privadas / públicas / loopback  
2. 🟡 Nivel intermedio → Clases de red (A, B, C)  
3. 🔴 Nivel avanzado → Subnetting básico (máscara de red)

---

## 🏆 Ranking

El juego guarda las mejores puntuaciones localmente en el navegador usando `localStorage`.
